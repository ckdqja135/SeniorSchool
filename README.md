##### Public : Bootstrap, EJS, CSS <br />
##### Front : Vanilla JS <br />
##### Back : Node JS, Express, Sequelize <br />
------------------------------------------

Frontend서버 실행 순서
1. Frontend 디렉토리 이동
``` shell
  cd Frontend
```

2. 서버 실행
``` shell
  nodemon ./bin/www
```

Backend서버 실행 순서
1. Backend 디렉토리 이동
``` shell
  cd Backend
```

2. 서버 실행
``` shell
  nodemon ./bin/www
```
